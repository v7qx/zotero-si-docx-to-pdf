# Zotero SI Docx to PDF

一个 Zotero 插件，用来把条目中的补充材料 Word 附件转换为 PDF。

当条目下新增符合条件的 `.doc` 或 `.docx` 子附件时，插件会调用 Microsoft Word 或 LibreOffice 进行转换，并将生成的 PDF 添加回原条目。

A Zotero plugin for converting Supplementary Information Word attachments into PDF.

When an eligible `.doc` or `.docx` child attachment is added to an item, the plugin uses Microsoft Word or LibreOffice to convert it and adds the generated PDF back to the same item.
